# 🤖 AI Resume Analyzer

An AI-inspired web application that analyzes resumes and calculates an ATS-style skill score.

## Features

- Resume Skill Analysis
- ATS Score Calculation
- Missing Skill Detection
- Learning Suggestions
- Interactive Progress Bar

## Technologies

- HTML5
- CSS3
- JavaScript

## Future Enhancements

- PDF Resume Upload
- OpenAI Integration
- Job Description Matching
- Resume Recommendations

## Demo

Paste a resume and instantly receive:

- Resume Score
- Found Skills
- Missing Skills
- Learning Recommendations
